String:
It is a class in java which is used to store sequence of characters.
we can create string objects by following ways:
1. using string literal
syntax:  String  s1 =  " ";
- these string will be stored in string constant pool area inside heap memory.

2. using new keyword
syntax:  String  s2  =  new  String(" ");
- these will be stored in direct heap area.
