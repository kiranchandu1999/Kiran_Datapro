CSS Positions

static
- It's default position of every element
relative
- It's positioned relative to it's original position and it occupies original space
absolute
- It's positioned relative to it's nearest positioned ancestor and it doesn't take any space
fixed
- It's positioned with respect to view port and it doesn't take space
sticky
- It's combination of relative and fixed
