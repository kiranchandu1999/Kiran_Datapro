Multi dimensional array:
when an array contains multiple arrays in it then it is called multi-dimensional array.

syntax for 2D:
data_type[ ][ ]  variable_name  =  new  data_type[ rows ][ cols ];
ex:  4, 5, 8, 6
        1, 3, 7, 2
         9, 4, 5, 1
int[ ][ ]  arr1  =  new int[3][4];

// data read and write
arr1[1][2] ---> 7
