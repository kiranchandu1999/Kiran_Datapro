Control statements:

- if statement
syntax:
if (condition) {
      // statements
}

- if else
syntax:
if (condition) {
      // statements
}
else {
     // statements
}

- if else if ladder
syntax:
if (condition) {
      // statements
}
else if (condition) {
     // statements
}
else {
     // statements
}
